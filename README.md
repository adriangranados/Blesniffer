# Buetooth LE sniffer (Beta)

A Buetooth LE sniffer for CC2540 USB dongle and macOS.

This project is __incomplete__. The application can not be saved in files in PCAP format yet. :-(

## Requirements

* Texas Instruments CC2540EMK-USB
* macOS 10.12
* Xcode 8.1
